A Cake Landing Pages Showcase Study

April 26th, 2020

After walking a weekend in Newport, RI, it was noticeable how small family-owned touristy places seem to be struggling more than others. However, they devised different possibilities to expand their business such as the “pastry delivery home” effort and order by phone. Small places are even delivering for a few dollars in order to resist. Especially in a city like Newport where tourism is the heart of the business, it is even more important to provide as much effort as possible to local business that in the middle of the winter hit twice since the pandemic started. It is even more important for them to show off and make customers come to their business weather for a simple coffee or for an elaborate and detailed creation such as a cake. The proposed landing page could be used as an initial and extremely inexpensive example of a local website. A lot of functionalities are shown in such a little page that can give any customer a sensation of eating at “home” or whether to use a pick up service, or a curb pick up, very fashionable lately. The solution below shows a simple navbar which quickly connects to any specific session of the landing page, a quick collection of the type of cakes that the business is able to make and a booking option. More importantly it is shown a cart where to add the products that customers would want to buy. Possibility of terase via responsive buttons and a cool contact us option that allows to drop a quick note to the business. This is a simple proposal that hopefully can lay-out potential ideas on how to expand it. As a quick note it should be mentioned that not necessarily it needs to be pro-software programmers or engineers for this type of creation but enough understanding of JavaScript and HTML would suffice to satisfy very small businesses or micro businesses from an economical point of view. // show cart
// declaration of immeditely invoked function expression


(function() {
  const cartInfo = document.getElementById('cart-info');
  const cart = document.getElementById('cart');


  cartInfo.addEventListener('click', function() {
    cart.classList.toggle('show-cart');
  })
})();


// add items to the cart
(function() {
  const cartBtn = document.querySelectorAll('.store-item-icon');


  // looping through all the cart icons on the images
  cartBtn.forEach(function(btn){
    btn.addEventListener('click', function(event){
      if(event.target.parentElement.classList.contains('store-item-icon')){


        let fullPath = event.target.parentElement.previousElementSibling.src;
        // the method indexOf will help find the position of the image
        let pos = fullPath.indexOf('img') + 3;
        // if we use fullPath.indexOf('img'); we obtain img/cake-2.jpeg
        // if we use fullPath.indexOf('img') + 3; we obtain /cake-2.jpeg
        // we want the second because we are trying to get to the index of the image


        let partialPath = fullPath.slice(pos);


        const item = {};
        item.img = `img-cart${partialPath}`;


        let name = event.target.parentElement.parentElement.nextElementSibling.children[0].children[0].textContent;
        item.name = name;
        let price = event.target.parentElement.parentElement.nextElementSibling.children[0].children[1].textContent;
        let finalPrice = price.slice(1).trim();
        item.price = finalPrice;


        const cartItem = document.createElement('div');
        cartItem.classList.add(
          'cart-item',
          'd-flex',
          'justify-content-between',
          'text-capitalize',
          'my-3');


        cartItem.innerHTML = `
          <img src="${item.img}" class="img-fluid rounded-circle" id="item-img" alt="">

            <p id="cart-item-title" class="font-weight-bold mb-0">${item.name}</p>

              <span id="cart-item-price" class="cart-item-price mb-0">${item.price}</span>
